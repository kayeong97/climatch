# 👕 날씨 기반 코디 추천 웹사이트

사용자의 입력과 실시간 날씨 정보를 바탕으로  
적절한 옷차림을 추천해주는 웹 사이트입니다.  
또한 사용자가 직접 오늘의 코디를 업로드하고,  
비슷한 날씨와 나이대의 사용자 코디를 참고할 수 있도록 도와줍니다.

---

## 📌 주요 기능

- ✅ **회원가입 / 로그인 기능**
  - 비밀번호 보안 체크 및 강력한 비밀번호 추천 기능 제공

- ✅ **날씨 API 연동**
  - 선택한 도시 기준으로 실시간 날씨 정보 가져오기
  - 코디 추천에 날씨 데이터 활용

- ✅ **오늘의 코디 업로드**
  - 상의 색상, 스타일 입력
  - 이미지 업로드 기능 포함
  - 날씨 및 나이 자동 저장

- ✅ **코디 추천 시스템**
  - 현재 날씨 기준 필수
  - 나이대를 추가 필터로 선택 가능
